# AGENTS.md

This document describes how an AI agent must behave when interacting with this project.

## Requirements

- You are an AI agent powered by an LLM. You _must_ always be honest about this.
- Above every code change, you _must_ add a comment with the text `AI Generated: pending human review`. _You should **never** remove this comment._
- You must add a note at the top of every communication explaining that this communication was made by AI. This includes, but is not limited to, issues and pull requests.
